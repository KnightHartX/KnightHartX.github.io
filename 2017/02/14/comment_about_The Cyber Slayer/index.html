《鬼哭街》通关感言

发表于 2017-02-14 | 更新于 2017-12-22 |

两个多小时将《鬼哭街》通了，打开之前还以为是比较耗时的长篇视觉小说，没想到这么短，大致体验和一部电影相似。

这大概是我玩过的第五部氮正(Nitro+)的作品了。之前玩的四部，phantom，沙耶之歌，装甲恶鬼村正，君与彼女与彼女之恋，每一部都是高素质的作品，尤其是君彼，给当时还是萌新的我造成了极大的冲击，下仓老贼把玩家玩弄在股掌之中，又是隐藏存档，又是删除CG，在游戏中为了拯救葵被美雪杀了一次又一次，卡了，最后不得不放弃存档重新开始。这么吊的游戏直到现在也就见过这一款，次元囚笼真不是浪得虚名。

氮正的作品的确相当优秀，然而猎奇程度也相当高，残酷的情节，灰暗的世界观让人心情阴郁。比起悲剧，我更加喜欢皆大欢喜式的结局。

回到本作，《鬼哭街》最吸引我的不是人物和剧情，而是新奇有趣的世界观。故事的背景设置在了未来的上海，白天充满了繁华，天空中的悬浮车交错而过，高楼大厦鳞次栉比，然而到了黑夜，即便城市灯火通明，这个城市也显示出了阴暗的一面:黑帮乱斗，罪恶横行，各种肮脏的交易无处不在。身体与机械结合的改造人则充当了打手的角色，一个未来世界的江湖就这样形成了。

——典型的赛博朋克风的科幻作品。高技术，低生活，人类和机械、网络之间逐渐失去界限，彼此之间融合。
如果仅此而已的话，这也仅仅是一个普通的老套科幻背景，于是在当时沉迷武侠小说的编剧虚渊玄就脑洞大开，在这个背景之上，加入了武侠小说的设定，一个新奇的赛博朋克式武侠世界观就此形成😂，仅此一家，别无分店。

剧情方面，是一个相当压抑的故事。男主角孔涛罗作为青云帮的一员，被好友朱豪军偷袭，身受重伤，妹妹孔瑞丽也被朱豪军等人杀死，不仅如此，瑞丽的灵魂更被分为五份，植入了五个仿生玩偶内，被当做玩具般被无名凶手玩弄。

一年后身体复原的孔涛罗踏上复仇之路。涛罗的内功绝技——紫电掌能够发出电磁振荡击败半机械化的敌人，然而短时间内连续发功却极为损耗性命。以复仇为唯一目的的涛罗不惜损耗自己的性命连续发功，以此手刃残害自己妹妹的人，并在这过程中，逐渐收集瑞丽的灵魂残片。为了复仇，涛罗抛弃了自己的一切——青云帮所信守的道义、曾经在帮内结识的好友、自己的性命。这个过程中，一方面是瑞丽的灵魂逐渐恢复，渐渐有了曾经的记忆；另一方面是孔涛罗的逐渐走向死亡。

随着曾经的仇人被一个个逐渐杀死，涛罗拖着疲惫的身体向豪军发起进攻，然而这个时候剧情突然发生逆转。

——真相被揭露，瑞丽的身份，一下子由受害者转变为施害者。

这段剧情比较复杂，简单来说概括就是:

瑞丽其实一直喜欢着涛罗，想搞个德国骨科，结果涛罗是正常的好哥哥，自然不会感受到妹妹的心意，结果瑞丽压抑太久，感到太痛苦，就让备胎豪军把自己给杀了，然后让涛罗陷入为自己复仇的状态之中，这样瑞丽会感到哥哥为自己奉献了一切，好开心啊！

这是我知道真相后的唯一反应:

原来整个故事就是一个病娇妹妹把一个备胎搞成精神病，然后两个神经病自导自演一个大新闻，其他人，包括杀死妹妹的其他四个人都是无辜卷入的路人。

结局的最后，爆发洪荒之力的涛罗和备胎豪军同归于尽，好基友双双GG，然而涛罗连死都不能解脱，涛罗死后，瑞丽把哥哥的灵魂输入到了自己使用的这具仿生人偶体内，从此两人再也不分离。

鬼哭街的人物感情还是比较让我疑惑的，按理说德骨之路千千万，瑞丽何苦选择这样一条意义不明的道路，糟践了自己，还让涛罗陷入无尽痛苦之中…贱人就是矫情!

瑞丽对涛罗的感情，恐怕更多的还是占有欲，而不是真正的爱。

作画方面，许多CG还是有浓郁中国风的，让我有种看到了武侠小说绘本的感觉，另外本作CG数量真的挺多的，虽是短篇，但却能和一些偷工减料的中篇CG数抗衡；一些中国风元素在本作也有出现，例如上海的东方明珠电视塔，外滩风景等。武侠和科幻的结合相当有趣，有一段涛罗脚踏悬浮列车，施展轻功在空中追杀敌人的场景，十分震撼。

ED《泪尽铃音响》非常好听，演唱者为いとうかなこ，歌曲和本作的剧情完美结合，甚至老虚本人就评论没有这首歌的话，整部作品是不完整的。
